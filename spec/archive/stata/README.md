# `spec/archive/stata/`

This directory holds Stata material that remains part of the repository but is not part of the
active paper-critical lane.

The partition is intentional:

- [`../../stata/`](../../stata/)
  - active paper-critical Stata specs, exporters, and support files
- `analysis_branches/`
  - non-paper analysis branches kept for reference and future recovery
- `scratch/`
  - exploratory or non-canonical Stata work
- `archive/`
  - older retired Stata layers and historical log folders

Nothing here is deleted; it has simply been moved out of `spec/stata/` so the active empirical
surface is easier to follow.
